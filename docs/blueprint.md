## Core Features:

- Expense Entry: Allows users to input their monthly expenses, including value, group, and payment method.
- Expense Categorization: Enables grouping of expenses into predefined categories (e.g., food, rent, transport) for better analysis.
- Payment Method Tracking: Tracks different payment methods used (e.g., credit card, debit card, cash) for each expense.
- Recurring Expense Management: Handles expenses paid in installments by distributing the payment across the corresponding months automatically.
- Expense Editing: Allows editing of expenses, even for installments. Updates the values automatically to future months.
- Expense Deletion: Allows permanent deletion of individual or recurring expenses.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to create a calm and trustworthy atmosphere.
- Background color: Very light blue (#F0F8FF).
- Accent color: Light green (#90EE90) for positive actions like adding an expense, or for categories of income.
- Body and headline font: 'PT Sans' for readability and a modern feel.
- Use clear and simple icons for expense categories and payment methods.
- Clean and intuitive layout for easy navigation and expense management.
- Subtle animations for adding/deleting expenses.
